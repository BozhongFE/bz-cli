# bz-cli

一个播种前端命令行工具，详细请看[bz-cli-wiki](https://github.com/BozhongFE/bz-cli/wiki)

## 使用方法

```bash
# 默认最新代码，不推荐
npm install -g https://github.com/BozhongFE/bz-cli
# tag 形式安装，推荐，请自行查看最新的 tag 版本号
npm install -g https://github.com/BozhongFE/bz-cli#v1.0.0
bz <command> [options]
```

## 命令列表

```bash
bz config
bz less
bz css
bz js
bz html
bz build
```
